## Описание

В этом репозитории хранится код для трансляции языка Neniy

## Сборка

Собрать можно пока только под 1.21.11  

Сначала нужно собрать Neniy и только после этого собирать эту библиотеку

Debug-сборка:
```
make mc_version=1.21.11
```

Release-сборка:
```
make mc_version=1.21.11 build=release
```

В директории с переданной версией появляется `impl.so` - динамическая библиотека, которая также копируется в `$HOME/Neniy/asset/1.21.11` (предполагается, что проект [Neniy](https://github.com/gh-Neniy/Neniy) слонирован в домашнюю директорию)
